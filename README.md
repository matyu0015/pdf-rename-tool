# PDF名前変更ツール

PDFファイルの中身を見ながら、簡単に名前を変更できるWebアプリケーションです。
すべての処理はブラウザ内で完結し、サーバーにデータは送信されません。

## 特徴

- **完全クライアントサイド処理**: すべてのデータはブラウザ内で処理され、外部サーバーに送信されません
- **PDFプレビュー機能**: PDFの中身を確認しながら名前を選択できます
- **エクセル連携**: エクセルファイルから新しいファイル名のリストを読み込めます
- **一括ダウンロード**: 名前変更したPDFをZIPファイルで一括ダウンロード可能
- **レスポンシブデザイン**: PC・タブレット・スマートフォンに対応

## 使い方

### 1. エクセルファイルの準備

新しいファイル名を記載したエクセルファイルを準備してください。

- **形式**: `.xlsx` または `.xls`
- **記入場所**: A列に新しいファイル名を記入
- **例**:
  ```
  A列
  ─────────────
  契約書_2024_01
  請求書_山田商事
  見積書_ABC株式会社
  報告書_第1四半期
  ```

### 2. Webアプリの使用手順

1. **エクセルファイルをアップロード**
   - 「1. エクセルファイルをアップロード」セクションでファイルを選択
   - A列のファイル名リストが表示されます

2. **PDFファイルをアップロード**
   - 「2. PDFファイルをアップロード」セクションで複数のPDFファイルを選択
   - 各PDFのプレビューが表示されます

3. **名前を選択**
   - 各PDFのプレビューを確認しながら、ドロップダウンから新しい名前を選択
   - 名前を選択すると「ダウンロード」ボタンが有効になります

4. **ダウンロード**
   - 個別ダウンロード: 各PDFの「ダウンロード」ボタンをクリック
   - 一括ダウンロード: 「すべてのPDFをZIPでダウンロード」ボタンをクリック

## ローカルで使用する方法

### 方法1: ブラウザで直接開く

1. `index.html` をダブルクリックしてブラウザで開く

### 方法2: ローカルサーバーで起動

Pythonがインストールされている場合:

```bash
# Python 3の場合
python3 -m http.server 8000

# ブラウザで http://localhost:8000 にアクセス
```

Node.jsがインストールされている場合:

```bash
# http-serverをインストール（初回のみ）
npm install -g http-server

# サーバー起動
http-server

# ブラウザで http://localhost:8080 にアクセス
```

## オンラインで公開する方法

### GitHub Pagesで公開（推奨・無料）

1. GitHubでリポジトリを作成
2. このフォルダの内容をpush
3. Settings → Pages → Source を "main" ブランチに設定
4. 数分後にURLが発行されます（例: `https://username.github.io/repository-name/`）

### Netlifyで公開（無料）

1. [Netlify](https://www.netlify.com/) にサインアップ
2. 「Add new site」→「Deploy manually」を選択
3. このフォルダをドラッグ&ドロップ
4. 即座にURLが発行されます

### Vercelで公開（無料）

1. [Vercel](https://vercel.com/) にサインアップ
2. 「Add New」→「Project」を選択
3. GitHubリポジトリを連携するか、フォルダをアップロード
4. 自動的にデプロイされます

## セキュリティについて

- すべての処理はブラウザ内で完結します
- PDFやエクセルファイルは外部サーバーに送信されません
- データはブラウザのメモリ上でのみ処理され、永続的に保存されません
- ページを閉じると、すべてのデータは削除されます

## トラブルシューティング

### PDFが表示されない
- ブラウザが最新版か確認してください
- ChromeまたはFirefoxを使用してください

### エクセルファイルが読み込めない
- ファイル形式が `.xlsx` または `.xls` であることを確認してください
- A列にデータが入力されていることを確認してください

### ダウンロードできない
- ポップアップブロックが有効になっていないか確認してください
- ブラウザのダウンロード設定を確認してください
